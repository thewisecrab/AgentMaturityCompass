# QUALIFICATION — IMMEDIATE CLOSE MOTION

## Purpose
Qualify fast, disqualify cleanly, and move only high-probability deals into close sequence within 24–72 hours.

## Core Framework: PACT-DO (0–12)
Score each category 0, 1, or 2.

- **P — Problem Severity**
  - 0: Nice-to-have pain
  - 1: Clear pain but tolerated
  - 2: Pain is costly/visible now (revenue loss, delays, compliance/trust risk)
- **A — Authority Access**
  - 0: No decision-maker path
  - 1: Influencer only, DM uncertain
  - 2: Direct access to budget owner + champion
- **C — Commitment Timeline**
  - 0: “Maybe this quarter”
  - 1: Targeted this month, weak urgency
  - 2: Decision in ≤14 days; start in ≤30 days
- **T — Technical Readiness**
  - 0: No stack clarity / no owner
  - 1: Stack known, implementation owner unclear
  - 2: Stack + owner + integration path confirmed
- **D — Data Availability**
  - 0: No baseline metrics/data access
  - 1: Partial baseline, delayed access
  - 2: Baseline + instrumentation accessible now
- **O — Outcome Urgency**
  - 0: No quantified target
  - 1: Target exists, low accountability
  - 2: Quantified target tied to executive priority

## Stage Rules
- **0–5:** Disqualify or nurture (do not send proposal)
- **6–7:** Hold in discovery; require gap-closing action
- **8–9:** Advance to solution + commercial call
- **10–12:** Immediate close motion (same-day proposal + decision plan)

## Mandatory Exit Criteria (before proposal)
All must be true:
1. Business problem stated in one sentence with impact.
2. Budget owner identified by role/name.
3. Decision process + date documented.
4. Success metric and baseline agreed.
5. Implementation owner identified.

## Red Flags (Auto-DQ unless corrected)
- Wants “free strategy” without buying intent.
- No decision-maker access after 2 attempts.
- Demands guaranteed revenue claims.
- Scope is undefined and refuses discovery discipline.
- Timeline mismatch: wants start now, but no internal owner.

## Discovery Questions That Drive Close
1. What breaks today if this isn’t fixed in 30 days?
2. Who signs and who can block this decision?
3. What metric moves first if we win in 14 days?
4. What systems/data must we touch in week 1?
5. What date do you want this live, and why that date?

## Commercial Readiness Checklist
- [ ] PACT-DO score documented
- [ ] Buying committee map captured
- [ ] Offer-path selected (Sprint / Pilot / Retainer)
- [ ] Pricing anchor shared verbally
- [ ] Objections logged and addressed
- [ ] Mutual action plan with dates agreed

- `target_metric`
- `implementation_owner`
- `offer_path`

## Close Trigger
If **PACT-DO ≥10** and all exit criteria are met:
- Send tailored offer within 2 hours.
- Book decision call within 48 hours.
- Include start date and onboarding kickoff slot.
