## Content Briefs by Format

### POV Posts (LinkedIn) — Template
```
Opening hook: [Observation or counterintuitive statement — 1 sentence]
Build: [3-4 lines expanding the insight with specifics]
Implication: [1-2 lines on what this means for the reader's situation]
Soft ask: [CTA — no hard sell]
```

### Threads (X/Twitter) — Template
```
Tweet 1: Hook — the insight or tension in one line
Tweets 2-4: Each expands one specific point (concrete, numbered)
Tweet 5: Takeaway + CTA link
```

### Carousels (LinkedIn) — Template
```
Slide 1: Hook / tension
Slides 2-4: Core content / breakdown
Slide 5: Proof / evidence (compliant framing)
Slide 6: CTA
```

### Proof Posts (LinkedIn) — Compliance Requirement
> Always frame as: "teams at this stage commonly…" or "we observe…" — never "our client got X result."
> Include: "Patterns described are directional and based on observed assessments. Results vary by context."
