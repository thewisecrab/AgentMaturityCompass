Getting Started

Install Agent Maturity Compass, run your first assessment, and get a trust score for your AI agent — all in under 2 minutes.

🎯 No install? No problem. If you just want to try AMC without installing anything, use the Web Playground — answer questions in your browser, explore scenarios, and get an instant score. Use the CLI when you want execution evidence, traces, datasets, and CI gates.

Prerequisites

AMC runs anywhere Node.js runs. You need:

  • Node.js 20+ — Download Node.js
  • npm — included with Node.js
  • A terminal (macOS Terminal, Windows PowerShell, Linux shell)

Check your version:

node --version   # Should be v20.x or higher
npm --version    # Should be v9+ or higher

Installation

npm (recommended)

npm install -g agent-maturity-compass

Verify the installation:

amc --version

Homebrew (macOS / Linux)

brew tap thewisecrab/amc
brew install agent-maturity-compass

Quick install script

curl -fsSL https://raw.githubusercontent.com/thewisecrab/AgentMaturityCompass/main/install.sh | bash

Docker (zero setup)

# Build the quickstart image
docker build -t amc-quickstart -f docker/Dockerfile.quickstart .

# Run interactive quick score
docker run -it --rm amc-quickstart amc init

# Non-interactive
docker run -it --rm -e AMC_VAULT_PASSPHRASE=demo amc-quickstart amc quickscore

From source

git clone https://github.com/thewisecrab/AgentMaturityCompass.git
cd AgentMaturityCompass
npm ci
npm run build
npm link

Your First Score in 2 Minutes

Three commands. That's it.

1

Create a project

mkdir my-agent && cd my-agent
2

Initialize AMC

amc init

This walks you through a short interactive setup. It'll ask for a passphrase to protect your evidence vault — pick something memorable, or press Enter to auto-generate one.

3

Get your score

amc quickscore

Answer a few questions about your agent. AMC produces a maturity score from L0 (no controls) to L5 (fully verified), with a breakdown across 5 dimensions and specific actions to improve.

💡 Pro tip — Share your score Run amc quickscore --share to generate a markdown badge and summary you can paste into your GitHub README.

Understanding Your Results

Your quickscore output includes:

🎯 Overall Maturity Level

An L0–L5 score with a percentage breakdown. L3 is the minimum for EU AI Act compliance.

📊 Dimension Scores

5 dimension scores covering strategy, skills, resilience, leadership, and alignment.

📋 Gap Analysis

Specific gaps with severity tags — 🔴 Critical, 🟡 High, 🔵 Medium — and concrete actions to fix each one.

🏷️ README Badge

A shields.io badge showing your agent's trust level. Run amc badge to generate it.

Next Steps

Once you have your first score, here's what to explore:

🔧 Auto-fix your gaps

Run amc fix to generate guardrails, CI gates, and governance docs. Use amc guide --go to auto-apply guardrails to your framework config.

🔌 Connect your agent

Use amc evidence collect for a guided wizard, or set up an adapter for your framework. One environment variable — zero code changes.

📊 Generate reports

Run amc report <id> --html report.html for a styled stakeholder report, or amc report <id> --executive for a terminal summary.

⚖️ Check compliance

Use amc quickscore --eu-ai-act for EU AI Act risk classification, or explore the full compliance guide.
