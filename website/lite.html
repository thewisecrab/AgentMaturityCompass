⚡ How It Works

1. Connect

Run amc init in your project. AMC sets up a proxy between your agent and the AI API. One environment variable, no code changes.

2. Observe

AMC captures every API call, tool use, and decision with cryptographic signatures. The agent can't forge its own evidence.

3. Score

140 diagnostic questions across 5 trust dimensions produce an L0–L5 maturity score. Cryptographically sealed and verifiable offline.

❓ FAQ

What is AMC?

Agent Maturity Compass is an open-source framework for scoring AI agent trustworthiness from execution evidence — not self-reported claims. It uses cryptographic evidence chains and 140 diagnostic questions.

What is the 84-point gap?

When scored using keyword matching, an AI agent scores 100/100. When AMC evaluates the same agent using execution-verified evidence, the real score is 16/100. That 84-point gap is documentation inflation.

Is AMC free?

Yes. MIT licensed, fully open source. Free forever.

Do I need to code?

No! Use the playground to score your agent by answering 15 questions. No installation needed.
