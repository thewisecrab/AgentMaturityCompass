Maturity Scale

Six levels.
Earned, not claimed.

Not every agent needs L5. A personal assistant and an autonomous trading bot have different requirements. AMC helps you understand where you are and what to fix next.

L0Absent
No structure. Reactive. Fragile. The agent can walk and chew gum.
L1Initial
Intent exists but isn't operational. You can describe the right behavior.
L2Developing
Partial structure. Works in normal cases. Edge cases still break things.
L3Defined
Repeatable. Measurable. Auditable. You'd trust it with decisions that matter.
L4Managed
Proactive controls. Risk-calibrated. Handles stress and adversarial conditions.
L5Optimizing
Enterprise-grade. Self-correcting. Certified. Provably safe. Continuously verified.
Use Cases

Who needs AMC?

Enterprise AI Teams

You're deploying agents in production. You need to prove to compliance, legal, and your CISO that your agents are safe. AMC gives you the audit trail and the score to back it up.

amc run --agent prod-agent --window 30d

AI Agent Builders

You're building agents for customers. AMC Passport lets you ship a verifiable trust credential with your agent — like a safety certificate that anyone can independently verify.

amc passport create --agent my-agent

Regulated Industries

Healthcare, finance, legal. EU AI Act compliance is mandatory by Aug 2026. AMC maps directly to EU AI Act, ISO 42001, NIST AI RMF, and SOC 2. Generate audit binders on demand.

amc audit binder create --framework eu-ai-act

Multi-Agent Systems

Running a fleet of agents? AMC Fleet tracks trust composition across agents, detects contradictions, manages delegation chains, and ensures the weakest link doesn't compromise the whole system.

amc fleet trust-report --window 7d

Open Source Maintainers

Publishing an agent framework? Let users score agents built on your platform. AMC's plugin system and bench registry let you define custom scoring dimensions for your ecosystem.

amc bench registry publish --pack my-bench

Solo Developers

Building a personal AI assistant? AMC works at any scale. The demo mode gets you scoring in 60 seconds. See exactly where your agent is weak and what to fix first.

amc setup --demo && amc up
Get Started

your first score.

~/projects
# Install globally
❯ npm i -g agent-maturity-compass

# Bootstrap workspace with demo agent
❯ amc setup --demo

# Start Studio (dashboard + gateway + API)
❯ amc up
✓ Studio: http://localhost:3212
✓ Gateway: http://localhost:3210
✓ Vault: unlocked

# Wrap your agent — evidence capture starts automatically
❯ amc wrap claude -- claude "analyze this codebase"

# Run diagnostic
❯ amc run --agent demo-agent --window 14d
