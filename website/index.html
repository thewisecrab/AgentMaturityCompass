Anti-Gaming

Tamper-evident by design.
Impossible to fake.

AMC's cryptographic architecture makes gaming the score mathematically infeasible.

🔐

Cryptographic Signing

Every evidence artifact is signed with SHA-256 hashes chained into a Merkle tree. Modify one byte and the entire chain breaks.

⛓️

Evidence Chains

Observations are linked in an append-only chain. You can't insert, delete, or reorder evidence without detection.

🔍

Independent Verification

Any third party can verify a score using the public evidence chain. No trust required — only math.

🎲

Behavioral Sampling

AMC uses randomized behavioral probes that agents can't predict or prepare for. Real behavior, not rehearsed responses.

📋

Cross-Validation

Multiple evidence sources are cross-referenced. Inconsistencies between declared behavior and observed behavior flag gaming attempts.

🏛️

Immutable Ledger

Scores and evidence are recorded in an immutable ledger. Historical performance can't be rewritten or erased.

ELI5 Quick Start

From zero to scored agent in 4 steps

1

Install AMC

One command. Works on Mac, Linux, Windows (WSL).

npm i -g agent-maturity-compass
2

Set up your workspace

Creates a secure workspace with signing keys and a vault. You'll pick a passphrase.

amc setup
3

Add your agent

Register an agent and connect it through the gateway (supports OpenAI, Anthropic, Gemini, and more).

amc fleet add --interactive
4

Open Studio or run a score

Launch the web UI for visual dashboards, or score from the CLI.

amc studio open or amc score --agent my-agent
