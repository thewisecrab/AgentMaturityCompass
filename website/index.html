Get Started

Four commands.
Zero to scored.

01

Install

Works on macOS, Linux, Windows (WSL). Python 3.10+ required.

pip install agent-maturity-compass
02

Setup

Creates workspace, generates signing keys, initializes encrypted vault.

amc setup
03

Score

Rapid 5-question assessment. Full 111-question diagnostic also available.

amc quickscore
04

Launch UI

Starts Studio with Console, Dashboard, Gateway, and REST API.

amc up
