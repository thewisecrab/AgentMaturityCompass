What makes this different_

Evidence-weighted scoring

Self-reported evidence is capped at 0.4× weight. Observed execution evidence scores at 1.0×. You can't game this with better documentation.

Cryptographic proof chains

Every score is backed by Ed25519 signatures and Merkle-tree evidence chains. Tamper-evident. Auditor-friendly. Not just a number in a dashboard.

Zero code changes

AMC wraps your existing agent with one environment variable. The agent doesn't know it's being observed. No SDK integration required.

481 CLI commands

Score, fix, shield, watch, comply, fleet, vault, passport — every trust workflow is a CLI command away. Scriptable. Composable. CI-ready.

14 framework adapters

LangChain, CrewAI, OpenAI, AutoGen, Claude Code, Gemini, LlamaIndex, Semantic Kernel, and 6 more. One tool for your entire stack.

Free forever
