# Founder-Led Sales Playbook (Close One $5k Sprint Fast)

## Goal
Close **one $5,000 sprint engagement** quickly with high-fit prospects by running a tight founder-led process: focused outreach, sharp discovery, clear qualification, fast proposal, decisive close.

---

## 1) Offer Definition (What You’re Selling)

### Core Offer
- **Productized Sprint**: 2-week delivery sprint
- **Price**: $5,000 fixed
- **Outcome**: one concrete business result (e.g., conversion lift, onboarding fix, revenue experiment, automation bottleneck removed)
- **Scope cap**: one core problem, one owner, one KPI

### Promise
- You do not promise miracles.
- You promise: **speed, focus, and decision-ready output**.

### Deliverables (example)
- Problem diagnosis + sprint plan (Day 1–2)
- Build/implementation (Day 3–9)
- QA + launch/readout (Day 10)
- Final handoff + recommended next steps

---

## 2) ICP + Trigger-Based Targeting

### Best-Fit Prospect Profile
- Founder-led SMB/startup with urgent growth or ops bottleneck
- Can make decisions without committee drag
- Has visible pain tied to money/time
- Can afford $5k from operating budget

### Trigger Events (prioritize these)
- New launch failed to convert
- Paid ads running but ROI weak
- Sales pipeline stalled
- Team repeatedly doing manual workflows
- Upcoming deadline (investor update, campaign, product launch)

### Red Flags
- “Just exploring” with no timeline
- No clear owner for this problem
- Wants full roadmap/retainer but no immediate priority
- Asks for free strategy without commitment

---

## 3) Founder-Led Pipeline (7-Day Close Loop)

### Day 0–1: Outreach
- Send 10–20 highly relevant messages/day
- Use short, specific hook around observed pain
- CTA: 20-min diagnostic call

### Day 1–3: Discovery Calls
- Run structured discovery (script below)
- Qualify in-call
- If fit: offer sprint path and reserve slot

### Day 3–4: Proposal + Close Call
- Send 1-page proposal same day
- Book 15-min close call to handle objections and secure yes/no

### Day 4–7: Decision + Start
- Follow next-step cadence
- Collect payment + kickoff date
- If no decision by deadline, close-lost and recycle later

---

## 4) Discovery Call Script (20–30 Minutes)

## A. Set Frame (2 min)
"Thanks for making time. I’ll ask a few questions to understand the problem, impact, and urgency. If there’s a strong fit, I’ll suggest a 2-week sprint. If not, I’ll say so directly. Sound good?"

## B. Current State + Pain (6–8 min)
- "What’s the main problem you want fixed right now?"
- "What have you tried so far?"
- "Where is this breaking in day-to-day operations/revenue?"

## C. Impact + Urgency (5 min)
- "What is this costing you weekly (money, time, missed deals)?"
- "Why solve this now vs next quarter?"
- "What happens if nothing changes in 30 days?"

## D. Decision Process (5 min)
- "Who decides on this?"
- "Is budget already available for fixing this fast?"
- "If we agree on scope, can you approve today/this week?"

## E. Fit Summary + Recommendation (3–5 min)
"Based on what you shared, the highest-leverage move is a 2-week sprint focused on [X], with success measured by [Y]. Fixed fee is $5,000. If useful, I can send a one-page plan today and hold a start slot for [date]."

## F. Commitment Ask (2 min)
- "Do you want me to send the sprint proposal and hold your slot?"
- If yes: schedule close call immediately (within 24h)

---

## 5) Close Criteria (Must-Haves)
Close only when all are true:
1. **Pain is clear and costly**
2. **Outcome is specific and measurable**
3. **Decision-maker is on the call (or directly accessible)**
4. **Budget for $5k is real**
5. **Timeline is now (start within 1–2 weeks)**
6. **Single owner exists on client side**

If any are missing, do not force close. Move to nurture or disqualify.

---

## 6) Objection Handling (Fast, Calm, No Pressure)

### Objection: “$5k is too expensive.”
Response:
"Totally fair. Compared to inaction, this is a short focused investment to solve [pain] now. If implementation goes well, you may recover cost through [specific upside], but outcomes vary by context. If budget is still a blocker, better to pause than run a half-committed sprint."

### Objection: “Need to think about it.”
Response:
"Makes sense. Usually the decision comes down to priority and ownership. What specifically do you need to think through—scope, budget, or timing?"

Then isolate and solve the real blocker.

### Objection: “Can you discount?”
Response:
"I keep sprint pricing fixed to protect delivery quality and speed. If needed, we can reduce scope but not the standards."

### Objection: “Send more details first.”
Response:
"Happy to. I’ll send a one-page plan, but it will stay high level until we confirm commitment and owner alignment."

### Objection: “We need internal alignment.”
Response:
"Great—let’s book a 15-min alignment call with all decision stakeholders so we can decide quickly either way."

---

## 7) Next-Step Cadence (After Discovery)

### If Strong Fit
- **Same day (within 2 hours):** send 1-page proposal + scope + timeline + payment terms
- **+24 hours:** follow-up with simple decision CTA: "Proceed / pause"
- **+48 hours:** final check-in + slot expiry reminder
- **+72 hours:** close-lost politely if no decision

### Follow-up Templates

**Proposal Send**
Subject: 2-week sprint plan for [Company]

"Great speaking today. As discussed, attached is the 2-week sprint plan to solve [problem], targeting [metric]. Fixed fee: $5,000. If you’d like to proceed, reply ‘go’ and I’ll send kickoff + payment link. I can hold a start slot until [date/time]."

**24h Follow-up**
"Quick one—do you want to move forward with the sprint, or should we pause for now? Either answer is fine; I just want to close the loop."

**48h Final Reminder**
"Final ping before I release the slot. If priorities changed, no issue—happy to revisit when timing is better."

---

## 8) Execution Discipline (Founder Rules)
- Keep calls short and diagnostic, not performative
- Don’t over-customize before commitment
- Don’t chase low-intent prospects
- Protect time blocks for outreach + calls + follow-ups daily
- One clear offer > many vague options

---

## 9) Weekly Scorecard
Track daily and review every Friday:
- Outreach sent
- Discovery calls booked
- Qualified opportunities
- Proposals sent
- Close rate
- Time-to-close (days)
- Revenue closed ($)

Target for fast first close:
- 40–60 targeted outreaches/week
- 5–8 discovery calls/week
- 2–3 qualified proposals
- 1 signed sprint ($5k)

---

## 10) Close-Lost to Recycle (No Burn)
When a deal stalls, end cleanly:
"Sounds like timing isn’t ideal right now. I’ll close this for now so we’re not in limbo. If [pain] becomes urgent again, reply and we can restart quickly."

This preserves trust and keeps pipeline clean.
