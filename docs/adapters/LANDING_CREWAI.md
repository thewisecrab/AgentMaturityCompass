# AMC for CrewAI Users

You already use CrewAI. AMC wraps it — zero code changes.

## Get started

```bash
amc wrap crewai -- python crew.py
amc quickscore
amc fix
```

- Execution evidence from real CrewAI runs
- Multi-agent trust scoring (Fleet workflows)
- Adversarial assurance packs
- CI gates for trust thresholds
