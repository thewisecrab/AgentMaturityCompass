# AMC for Any CLI Agent

AMC wraps any command-line agent — zero code changes.

## Get started

```bash
amc wrap generic-cli -- python my_bot.py
amc quickscore
amc fix
```

## What you get
- Execution evidence from any CLI-invoked agent
- Trust scoring across 5 dimensions
- 86 adversarial assurance packs
- CI gates, compliance outputs, and monitoring
