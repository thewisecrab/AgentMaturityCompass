# AMC for OpenAI Agents SDK Users

You already use the OpenAI Agents SDK. AMC wraps it — zero code changes.

## Get started

```bash
amc wrap openai-agents-sdk -- python my_agent.py
amc quickscore
amc fix
```

## What you get
- Execution evidence from real OpenAI agent runs
- Trust scoring from observed behavior, not claims
- Adversarial testing with 86 assurance packs
- CI integration for trust regression prevention
