# AMC for Claude Code Users

You already use Claude Code. AMC wraps it — zero code changes.

## Get started

```bash
amc wrap claude-code -- claude "analyze this code"
amc quickscore
amc fix
```

## What you get
- Execution evidence from real Claude Code sessions
- Trust scoring from observed behavior
- Adversarial assurance against prompt injection and data leakage
- CI-ready trust gates
